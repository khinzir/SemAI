# CSC111 — Digital Logic

> **Summary:** Learn the fundamentals of digital systems, Boolean algebra, combinational & sequential circuits, registers, counters, and memory. The course builds the foundation for computer architecture and digital hardware design.

**Credits:** 3 (Theory + Lab)

---

# Course Outline

### Unit 1 — Binary Systems *(6 Hrs.)*

* Number systems

  * Binary, Octal, Hexadecimal
  * Base conversions
* Binary arithmetic

  * Complements
  * Signed binary numbers
* Binary codes

  * BCD
  * 2421 Code
  * 84-2-1 Code
  * Excess-3
  * Gray Code
* Storage & Logic

  * Registers
  * Binary Logic

---

### Unit 2 — Boolean Algebra & Logic Gates *(5 Hrs.)*

* Boolean algebra fundamentals
* Laws and theorems
* Boolean functions
* Logic operations
* Logic gates
* Integrated Circuits (ICs)

---

### Unit 3 — Boolean Function Simplification *(5 Hrs.)*

* Karnaugh Maps (2, 3 & 4 variables)
* Product of Sums (POS)
* NAND implementation
* NOR implementation
* Don't-care conditions
* Prime implicants

---

### Unit 4 — Combinational Logic *(5 Hrs.)*

* Design procedure
* Adders & Subtractors
* Code converters
* Analysis procedure
* Multilevel NAND/NOR circuits
* XOR circuits

---

### Unit 5 — MSI & LSI Logic *(8 Hrs.)*

* Parallel adders & subtractors
* Decimal adder
* Magnitude comparator
* Decoder & Encoder
* Multiplexer
* ROM
* PLA
* PAL

---

### Unit 6 — Sequential Logic *(10 Hrs.)*

* Flip-Flops
* Triggering methods
* Clocked sequential circuits
* State equations
* State reduction
* Asynchronous circuits
* Latches

---

### Unit 7 — Registers & Counters *(6 Hrs.)*

* Registers
* Shift registers
* Ripple counters
* Synchronous counters
* Timing sequences
* Memory

---
